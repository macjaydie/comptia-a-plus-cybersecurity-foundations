# Basic Linux Commands



Linux command-line skills are essential for managing systems and troubleshooting.



---



## ls



Lists files and directories.



Examples:

- ls shows contents

- ls -l shows detailed view

- ls -a shows hidden files



---



## cd



Changes directories.



Examples:

- cd /home moves to the home directory

- cd .. moves up one directory



---



## chmod



Changes file permissions.



Example:

chmod 755 filename



Controls who can read, write, or execute files.



---



## chown



Changes file ownership.



Example:

sudo chown user:group filename



Used to assign files to users and groups.



---



## grep



Searches for text within files.



Example:

grep "error" logfile.txt



Useful for analyzing logs and configuration files.



---



## top



Displays running processes and system resource usage in real time.



Used to identify performance issues.
