# Basic Windows Commands



Windows command-line tools help IT professionals diagnose and manage systems.



---



## ipconfig



Displays IP configuration information.



Common uses:

- ipconfig /all shows detailed network settings

- ipconfig /release and /renew refresh IP address

- ipconfig /flushdns clears DNS cache



---



## ping



Tests connectivity to another device.



Example:

ping google.com



Helps determine if a host is reachable.



---



## tracert



Traces the path packets take to reach a destination.



Example:

tracert google.com



Used to identify where network delays occur.



---



## netstat



Displays active network connections and ports.



Common uses:

- netstat -an shows active connections

- netstat -b shows which programs are using connections



---



## tasklist and taskkill



Used to manage running processes.



Examples:

- tasklist shows running processes

- taskkill /PID processID ends a process
