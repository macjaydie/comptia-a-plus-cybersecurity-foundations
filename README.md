This repository documents my structured learning journey through CompTIA A+ topics while building foundational knowledge for a career in IT and Cybersecurity.



The goal of this repo is to:

- Reinforce core IT concepts

- Build strong troubleshooting skills

- Develop practical command-line knowledge

- Create a professional knowledge base for future reference



---



## Repository Structure



This repository is organized according to the main CompTIA A+ knowledge domains.



### 01 — Hardware

Covers internal computer components, peripherals, storage devices, and system hardware fundamentals.



### 02 — Operating Systems

Focuses on Windows and Linux basics, system tools, security concepts, and OS-level troubleshooting.



### 03 — Networking

Introduces networking fundamentals including IP addressing, common ports, protocols, and basic network security.



### 04 — Security

Explains essential cybersecurity principles such as authentication, authorization, malware types, and security best practices.



### 05 — Troubleshooting

Documents structured troubleshooting methods for hardware, operating systems, and networking issues.



### 06 — Scripting Basics

Covers basic command-line tools and scripting concepts used for system management and troubleshooting in Windows and Linux.



### 07 — Operational Procedures

Highlights professional IT practices including documentation, safety, communication, change management, and disaster recovery.
